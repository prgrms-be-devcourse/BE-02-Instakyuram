KDT Infra
===


### 초기 설정
---
>  setup 스크립트는 최초 한번 실행 해야 한다.


### 사용법

## 실행 스크립트
```
./mysql
./redis
./sonarqube
```

### Container 폴더 설명
* 종료 스크립트 
  * 각 **_continer 의 `stop` 스크립트 실행 
* 백그라운드 실행 스크립트
  * 각 **_continer 의 `start` 스크립트 실행
